Web A11y: A passion for inclusivity

Web accessibility is a passion of mine.

I have provided expertise and written code for numerous organizations and universities to secure their WCAG AA compliance, protecting them from legal troubles and extending their web presence to a larger and more diverse audience.

Going Native: Don't reinvent the wheel

When working with large development teams on 2-week sprint cycles, we have a short window to create robust UI components.

Using native and semantic HTML, we can eliminate hundreds of hours of development time in each sprint. In addition, we also spend less time making accessibility corrections thanks to the built-in accessibility features utilized by the OS, device, and web browser.
